# Programma Group REST MongoDB starter files

It has by default:
  * Express
  * MongoDB
  * Mongoose
  * Body Parser
  * Express Valudator
  * Mocha & Chai for testing
  * eslint Airbnb configuration
  * Swagger
  * Winston
  * Many more

To run:

```
$ npm install
$ npm start
```

Heavily based on the files by Wes Bos's course, [Learn Node](https://github.com/wesbos/Learn-Node)
